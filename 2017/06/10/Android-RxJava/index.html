關於 RxJava

RxJava 是一個 for Java VM 的 library，讓開發者用可觀察的序列來建構非同步作業及基於事件的程式。
使用它可以讓原本繁瑣的非同步作業程式碼變得簡潔、可讀性佳、容易維護，且讓線程的切換輕而易舉。

Observer Pattern

RxJava 中結合了幾個設計模式中的優點，其中有使用 Observer Pattern。
在 RxJava 中實現了 Observer pattern 兩個主要部分就是 Observable(可觀察對象) 及 Observer(觀察者)。
Observable(可觀察對象) 產生事件流，Observer(觀察者) 負責接收處理事件流，並透過 subscribe 讓兩者建立關係。

subscribe 多載

subscribe 方法是多載的

  • subscribe()
  • subscribe(Consumer<? super T> onNext)
  • subscribe(Consumer<? super T> onNext, Consumer<? super Throwable> onError)
  • subscribe(Consumer<? super T> onNext, Consumer<? super Throwable> onError, Action onComplete)
  • subscribe(Consumer<? super T> onNext, Consumer<? super Throwable> onError, Action onComplete, Consumer<? super Disposable> onSubscribe)
  • subscribe(Observer<? super T> observer)
